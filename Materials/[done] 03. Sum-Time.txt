Write a program to sum two time intervals given in format days::hours:minutes (days are optional).

Output:
 - Minutes should be displayed with leading zeroes.
 - If days are 0 in the results, don't display them.

Input:
1:15
1:25
Output:
2:40

Input:
1:30
2:35
Output:
4:05

Input:
0:06
23:55
Output:
1::0:01

Input:
3::11:30
12:45
Output:
4::0:15

Trivial: minutes % 60; hours % 24 + simple parsing
Level: 3/10
